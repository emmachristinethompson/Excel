# Excel Portfolio

This repository showcases a range of Excel skills through practical exercises and data analysis projects. The work covers spreadsheet design, data cleaning, formula development, reporting, visualisation, and analytical problem-solving using Microsoft Excel.

While I have extensive experience using Google Sheets in professional environments, this repository focuses specifically on demonstrating Excel proficiency and exploring Excel-specific tools and functionality.

## Skills Covered

- Data cleaning and preparation
- Formulas and functions
- Conditional logic
- Lookup functions
- Data validation
- Pivot tables
- Data visualisation
- Reporting and analysis
- Spreadsheet organisation and documentation

## Projects
